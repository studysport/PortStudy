Element

Short Definition

An element is a pure substance that consists of only one type of atom.

Detailed Explanation

Elements are the basic substances that make up all matter in the universe. Each element is made of atoms that have the same number of protons in their nucleus.

There are more than 100 known elements and each element has its own unique properties. Some common elements include hydrogen, oxygen, carbon, iron, and gold.

Elements are arranged in the periodic table according to their atomic numbers and chemical properties. Elements can combine with other elements to form compounds.
