Physics Formula Sheet (9th – 12th)

Class 9 Physics

Speed = Distance / Time
Velocity = Displacement / Time
Acceleration = (v - u) / t
Force = m × a
Momentum = m × v
Pressure = Force / Area
Work = Force × Distance
Power = Work / Time
Density = Mass / Volume

Class 10 Physics

Wave Speed = f × λ
Frequency = 1 / Time Period
Ohm’s Law: V = IR
Electric Power = VI
Efficiency = (Useful Energy / Total Energy) × 100
Transformer: Vp/Vs = Np/Ns
Lens Formula: 1/f = 1/v + 1/u
Magnification = Image Height / Object Height

Class 11 Physics

Vector Resultant: R = √(A² + B² + 2ABcosθ)
Newton Second Law: F = ma
Work = Fd cosθ
Kinetic Energy = ½mv²
Potential Energy = mgh
Centripetal Force = mv² / r
Stress = F / A
Young Modulus = Stress / Strain
Simple Harmonic Motion: T = 2π√(m/k)

Class 12 Physics

Coulomb Law: F = k(q₁q₂ / r²)
Electric Field = kq / r²
Electric Potential = kq / r
Capacitance = Q / V
Energy in Capacitor = ½CV²
Current = Q / t
Resistance = ρL / A
Magnetic Field = μ₀I / 2πr
Faraday Law: ε = - dΦ/dt
Induced EMF = BLv
RMS Voltage = V₀ / √2
Impedance = √(R² + (XL - XC)²)
Photoelectric Equation: hf = W + KE
Einstein Equation: E = mc²
De Broglie Wavelength = h / mv
Radioactive Decay: N = N₀e^(-λt)
Half Life = 0.693 / λ
