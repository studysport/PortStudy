RAM (Random Access Memory)

Short Definition

RAM (Random Access Memory) is a temporary memory of a computer that stores data and instructions while the computer is running.

Detailed Explanation

RAM (Random Access Memory) is one of the main components of a computer system. It is used to store data and programs temporarily that are currently being used by the CPU.

When you open any application such as a web browser, game, or software, the required data is loaded from the hard disk into RAM. The CPU processes this data quickly because RAM is much faster than secondary storage devices.

RAM is called volatile memory because it loses all stored data when the computer is turned off. That is why unsaved work is lost after shutdown.

The larger the RAM size, the more programs a computer can run smoothly at the same time without slowing down.

Key Points

  • RAM is temporary memory.
  • It is volatile memory.
  • It is faster than a hard disk.
  • It stores running programs and data.
  • More RAM improves system performance.
